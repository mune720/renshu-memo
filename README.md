# 練習メモ

フルート練習の気付きを書くための、iPhone向け超シンプルメモアプリ（PWA）。
「メモ」と「家計簿」の2タブ構成。ビルド不要の単一HTML。データは端末内（localStorage）のみに保存される。

## メモ（レッスンの気付き）

- **書く** — 開いてそのまま書くだけ。空行2つ（改行2回）で次のブロックになる（iPhone標準メモと同じ書き方）。
- **レッスン単位** — 左上のレッスン名をタップすると履歴一覧。「＋ 新しいレッスンを始める」で今のメモを保存して空の新規メモを開始。過去のレッスンは一覧から開いて見返せる。
- **時刻** — 各ブロックを「最初に書いた日時」が自動記録される。普段は非表示、「時刻」で表示切替。
- **整理** — 「整理」を押すと整理モードに：
  - 右の **＋** チップでカテゴリーを付けられる。「カテゴリー」表示で同じカテゴリーをまとめて見られる。「時系列」では日付ごとにまとまる。
- **コピー** — 1タップで今開いているレッスンを全文コピー（プレーンテキスト）。「時刻」表示がONのときは各ブロックの日時も一緒にコピーされる。Messengerに貼り付けて先生に送る。画面の文字をドラッグして一部だけ手動選択することもできる。

## 家計簿

- **入力** — 金額・区分（食費／交通費／学会費…）・メモ（任意）・日付を入力して「追加」。収入は扱わない。
- **区分** — 初期リストに加え「区分を編集」から自由に追加・削除できる（削除した区分の記録は「その他」へ移動）。
- **グラフ** — 月ごとに ①区分別の円グラフ ②月推移の棒グラフ ③区分別の月推移（積み上げ）を表示。月は ‹ › で移動。

## iPhoneへの入れ方

1. Safari で公開URLを開く
2. 共有ボタン → **「ホーム画面に追加」**
3. ホーム画面の「練習メモ」から起動（オフラインでも動く）

※ ホーム画面に追加して使うこと。Safariのタブで使い続けると、iOSの仕様で
7日間アクセスがないとデータが消されることがある（ホーム画面アプリは対象外）。

## 開発メモ

- `index.html` — 本体（HTML/CSS/JSすべてこの1ファイル）
- `sw.js` — Service Worker（オフラインキャッシュ。本体はネットワーク優先なので更新は自動反映）
- `manifest.webmanifest` / `icon-*.png` / `apple-touch-icon.png` — PWA用
- ローカル確認: このフォルダで `python3 -m http.server 8765` → http://localhost:8765
- データはブラウザの localStorage キー `renshu-memo-v1` に保存。サーバーには何も送らない。
