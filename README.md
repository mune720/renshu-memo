# 練習メモ

フルート練習の記録のための、iPhone向け超シンプルメモアプリ（PWA）。
1レッスンの中に「気付き」と「覚え書き」の2つの欄を持つ。ビルド不要の単一HTML。データは端末内（localStorage）のみに保存される。

## 気付き / 覚え書き（タブ切替）

1つの「レッスン」の中に、上部タブで切り替わる2つの欄がある。レッスンの切替・履歴・「時刻」表示は両欄で共通。

- **気付き** — 練習中の気付きを書く欄。カテゴリー分類・「整理」モードあり（下記）。
- **覚え書き** — レッスン後の覚え書き用の欄。シンプルに書くだけ＋コピー（カテゴリー・整理はなし）。

### 共通の操作

- **書く** — 開いてそのまま書くだけ。空行2つ（改行2回）で次のブロックになる（iPhone標準メモと同じ書き方）。
- **レッスン単位** — 左上のレッスン名をタップすると履歴一覧。「＋ 新しいレッスンを始める」で今のレッスン（気付き＋覚え書き）を保存して空の新規レッスンを開始。過去のレッスンは一覧から開いて見返せる。
- **時刻** — 各ブロックを「最初に書いた日時」が自動記録される。普段は非表示、「時刻」で表示切替（両欄共通）。
- **コピー** — 1タップで今表示している欄を全文コピー（プレーンテキスト）。「時刻」表示がONのときは各ブロックの日時も一緒にコピーされる。Messengerに貼り付けて先生に送る。画面の文字をドラッグして一部だけ手動選択することもできる。
- **整理（気付きのみ）** — 「整理」を押すと整理モードに。**＋** チップでカテゴリーを付けられる。「カテゴリー」表示で同じカテゴリーをまとめて、「時系列」では日付ごとにまとめて見られる。

## 家計簿（現在は非表示）

家計簿機能はコード・データを残したまま非表示にしている。`index.html` の `ENABLE_EXPENSE` を `true` にすればタブが復活する（金額・区分・支出メモの記録、月ごとの3グラフ＝区分別円／月推移棒／区分別積み上げ）。

## iPhoneへの入れ方

1. Safari で公開URLを開く
2. 共有ボタン → **「ホーム画面に追加」**
3. ホーム画面の「練習メモ」から起動（オフラインでも動く）

※ ホーム画面に追加して使うこと。Safariのタブで使い続けると、iOSの仕様で
7日間アクセスがないとデータが消されることがある（ホーム画面アプリは対象外）。

## 開発メモ

- `index.html` — 本体（HTML/CSS/JSすべてこの1ファイル）
- `sw.js` — Service Worker（オフラインキャッシュ。本体はネットワーク優先なので更新は自動反映）
- `manifest.webmanifest` / `icon-*.png` / `apple-touch-icon.png` — PWA用
- ローカル確認: このフォルダで `python3 -m http.server 8765` → http://localhost:8765
- データはブラウザの localStorage キー `renshu-memo-v1` に保存。サーバーには何も送らない。
